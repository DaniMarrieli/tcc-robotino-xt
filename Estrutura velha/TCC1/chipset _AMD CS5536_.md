**Chipset AMD CS5536:** Este é o **chipset auxiliar (_companion_)** do processador, confirmado no sistema do Robotino. Ele é responsável por gerenciar grande parte das funções de entrada/saída (I/O) de baixo nível do robô, como os controladores USB e as interfaces IDE para o armazenamento (Compact Flash). O kernel Linux precisa de _drivers_ específicos para o CS5536, como o `cs5535_gpio`, o que reforça a natureza legada e a dependência de um kernel antigo.
