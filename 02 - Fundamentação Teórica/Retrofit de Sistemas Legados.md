# Retrofit e Revitalização

## Conceito
O retrofit é o processo de modernização de uma plataforma de hardware antiga, mas mecanicamente funcional, visando combater a obsolescência tecnológica. No caso do Robotino XT, foca-se em substituir a inteligência (UCP) mantendo a base motora.

## Justificativa no Projeto
- **Sustentabilidade:** Evita o descarte de uma plataforma robótica robusta.
- **Custo-Benefício:** O upgrade para Raspberry Pi 4 é significativamente mais barato que a aquisição de um novo robô de porte similar.
- **Integração:** Permite que o robô de 2013 utilize sensores modernos (Lidar, Câmeras RGB-D) e frameworks atuais (ROS2).
