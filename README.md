# Revitalização e Adaptação de Sistema Embarcado Obsoleto em Plataforma Robótica Móvel: Um Estudo de Caso com o Robotino XT

## Sobre o Projeto

Este repositório reúne a documentação técnica, registros experimentais, referências bibliográficas, resultados e materiais de apoio desenvolvidos durante o Trabalho de Conclusão de Curso (TCC) intitulado:

**"Revitalização e Adaptação de Sistema Embarcado Obsoleto em Plataforma Robótica Móvel: Um Estudo de Caso com o Robotino XT"**

O trabalho investiga a modernização de uma plataforma robótica móvel Robotino XT, originalmente baseada em uma arquitetura computacional obsoleta, por meio da integração de uma nova unidade computacional baseada em Raspberry Pi 4 e tecnologias contemporâneas de desenvolvimento robótico.

---

## Objetivo Geral

Revitalizar e adaptar o sistema embarcado de uma plataforma Robotino XT, preservando sua infraestrutura eletromecânica original e criando uma base tecnológica compatível com ferramentas modernas de pesquisa e desenvolvimento em robótica móvel.

---

## Objetivos Específicos

- Recuperar e validar o funcionamento da plataforma Robotino XT.
    
- Analisar a arquitetura embarcada original baseada em AMD Geode LX800 e RTAI.
    
- Investigar os mecanismos de comunicação entre a CPU principal e a IO Board.
    
- Integrar uma plataforma Raspberry Pi 4 ao sistema legado.
    
- Desenvolver uma arquitetura híbrida para comunicação entre os componentes originais e modernos.
    
- Avaliar a viabilidade de utilização do ROS 2 em uma plataforma originalmente não projetada para esse ecossistema.
    
- Documentar desafios, limitações e oportunidades de modernização de sistemas embarcados legados.
